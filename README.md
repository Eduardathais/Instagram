# Instagram

Aplicação Instagram Clone - projeto com o objetivo de treinar e melhorar os conhecimentos em Ruby on Rails 

* Ruby version : 2.7.5

* Rails version : 6.1.7.4

* Features:
- Active Storage
- Action Cable
- Devise
- i18n
- Toastify

* Database : PostgreSQL
